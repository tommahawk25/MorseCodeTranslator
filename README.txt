Morse Code Translator
_____________________

This Morse Code Translator supports various Morse Code formats, it automatically identifies the format and translates it.

1) Run file translator.py in your editor with terminal, for example in PyCharm.
2) After you run the file, enter the message (text or Morse code) that you want to translate in terminal.
3) You get the result in terminal.

File test_translator.py is used for unit testing purposes.
